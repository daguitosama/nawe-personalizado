# Orders System

This system would be used by, the public site make orders flow, and the admin/order dashboard.

La orden tiene:

-   Fecha de creación
-   Cliente
-   Producto
    -   cantidad
    -   precio
    -   importe
-   Precio total
-   Estado:
    -   en venta
    -   produciendo
    -   entregada
    -   cancelada
-   Note (text)

En la página de producto, el formulario creara la orden. En el administrador, la orden podrá ser
modificada en

-   estado
